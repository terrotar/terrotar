## 📌 About Me

I'm an **Azure Data Engineer (Mid-Level)**, building and orchestrating ETL/ELT pipelines using:

- **Databricks + PySpark** – distributed processing & medallion architecture (bronze → silver → gold)
- **Azure Data Lake Gen2** – organized storage in Parquet/JSON
- **Azure DevOps** – CI/CD for data pipelines

I also have hands‑on experience with **Microsoft Fabric** (unified lakehouse, pipelines, analytics). My focus is delivering optimized tables for Power BI, enabling reliable business decisions.

**Complementary background:** Back‑end development with Python (Flask, FastAPI, unit tests) – a solid foundation for maintainable, efficient data pipelines.
