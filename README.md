## 📌 About Me – Data Engineering Focus

I'm an **Azure Data Engineer (Mid-Level)** currently working at **Dataside**. My daily work involves building and orchestrating ETL/ELT pipelines using:

- **Azure Data Factory** – ingestion from SQL databases and REST APIs
- **Databricks + PySpark** – distributed data processing and medallion architecture (bronze → silver → gold)
- **Azure Data Lake Gen2** – organized storage in Parquet/JSON
- **Azure DevOps** – version control and CI/CD for data pipelines
- **Azure Key Vault** – secure secrets management

I also have hands‑on experience with **Microsoft Fabric** (unified lakehouse, pipelines, and analytics). My goal is to deliver optimized tables for direct consumption in Power BI, enabling business decisions with reliable data.

**Complementary background:** Back‑end development with Python (Flask, FastAPI, unit tests), which gives me a solid software engineering foundation to write maintainable and efficient data pipelines.
